## Installation
1. Clone the repository
`
   https://github.com/BavKudhail/bav-react-portfolio.git
  `
2. Change the working directory
  `
  cd frontend
  `
3. Install dependencies
   `
  npm i
  `
4. Run the app
  `
  npm run start
  `
